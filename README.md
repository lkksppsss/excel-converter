# Excel 智慧格式轉換工具

> 純前端 SaaS — 讓任意格式的 Excel 轉成你要的格式

## 技術棧
- **框架**：React + Vite
- **Excel 處理**：SheetJS（xlsx.js）
- **拖曳連線**：待定
- **AI 欄位建議**：BYOK（用戶自帶 API Key，Claude / OpenAI）

## 快速開始

```bash
npm install
npm run dev
```
